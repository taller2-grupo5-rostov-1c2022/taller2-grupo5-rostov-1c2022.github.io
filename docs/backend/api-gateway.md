# API Gateway

## Servicio

Gateway a los micro-servicios de Spotifiuby.

Además de redireccionar el tráfico a los servidores correspondientes, provee las siguientes funcionalidades:

- Permitir **CORS**
- **Autenticar** y **autorizar** usuarios
